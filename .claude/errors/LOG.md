# Error Log

Errors are captured here by the `/0-log-error` command or manually.
When a pattern recurs 3+ times, promote it to PATTERNS.md.

## Format

```
### YYYY-MM-DD — [Error Title]
**Context:** What was being done
**Error:** The actual error message or behavior
**Root Cause:** Why it happened
**Fix:** How it was resolved
**Tags:** [category]
```
