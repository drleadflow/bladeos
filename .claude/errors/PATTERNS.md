# Error Patterns

Recurring errors (3+ occurrences) are promoted here from LOG.md.
Each pattern includes a prevention rule that should be followed
to avoid the error in the future.

## Format

```
### [Pattern Name]
**Frequency:** N occurrences
**Trigger:** What causes this error
**Prevention:** Rule to follow to avoid it
**Related Errors:** Links to LOG.md entries
```
